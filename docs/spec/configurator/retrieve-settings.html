Retrieve settings

GET 
/configurator/:version/configurator/settings

Settings that can be applied to the configurator such as theme preferences

Request​

Path Parameters

    version stringrequired

Query Parameters

    lang string

    (Optional) Language ISO

Responses​

  • 200

Success

  • text/plain
  • application/json
  • text/json
