# Get started with the Elfsquad APIs

Elfsquad provides a suite of APIs for integrating and extending Elfsquad's capabilities within your environment. Whether you're looking to enhance your ERP, CRM, CAD systems, build custom interfaces, manage quotations, or handle user identities across domains, our APIs offer solutions tailored for your needs.

See the [Authentication page](/docs/apis/authentication) for information about authentication.



<hr />

<h2 class="color-tech-savy-purple">Data API</h2>

<img src="/img/elfsquad-it-landscape.svg" height="200px" class="float-right" />
The Data API is your gateway to data within Elfsquad. It implements the OData specification, allowing for seamless querying and manipulation of all data. This API is perfect for integrating Elfsquad with your ERP, CRM, or CAD systems, ensuring that your data is always synchronized and up-to-date.

* Automatically synchronize product data and inventory levels between Elfsquad and your ERP system, ensuring that product information is always current and accurate across systems.
* Update customer records in your CRM system in real-time when changes occur in Elfsquad.
* Seamlessly push configuration details from Elfsquad to your CAD software to automatically generate or update design files based on customer specifications, streamlining the design process.

[Explore Data](/docs/apis/data-api)

<hr />

<h2 class="color-manufacturing-orange">Configurator API</h2>
<img src="/img/visual-configurator_2.webp" height="200px" class="float-right" />
Our Configurator API is designed to build custom user interfaces. This API provides possibilities for creating tailored configuration experiences that meet your unique business and product requirements.
<br />

* Develop configuration interfaces tailored to specific customer segments or product lines, offering a unique and guided product selection experience. 
* Integrate with 3D rendering engines to provide real-time visual feedback as customers configure products, enhancing engagement and decision-making.
* Create mobile applications that allow sales representatives or customers to configure products on-the-go, leveraging the configurator's capabilities for flexibility and convenience.

[Customize Now](/docs/apis/configurator-api)

<hr />

<h2 class="color-trust-turquoise">Quotation API</h2>
<img src="/img/quote.svg" height="200px" class="float-right" />
The Quotation API offers specialized endpoints for creating, managing, and retrieving quotation data within Elfsquad. This API streamlines the quotation process, making it easier to generate, update, and track quotations directly through your applications.

* Automatically generate detailed, accurate quotations based on triggers, reducing manual effort and increasing sales efficiency.
* Integrate with your CRM systems to seamlessly push quotation details from Elfsquad on every update.
* Develop systems to track the status of quotations, from creation to approval, providing sales teams with actionable insights to follow up effectively and close deals.

[Generate Quotes](/docs/apis/quotation-api)

<hr />

<h2 class="color-free-green">SCIM API</h2>
<img src="/img/channel-management.webp" height="200px" class="float-right" />
The SCIM (System for Cross-domain Identity Management) API provides endpoints for managing user identities within Elfsquad. This API simplifies user management across different systems and domains, enhancing security and reducing administrative overhead.

* Automate the provisioning and de-provisioning of user accounts across Elfsquad and other systems, ensuring access control is maintained efficiently as team dynamics change.
* Synchronize user identity information (e.g., roles, permissions, contact details) across multiple systems, ensuring consistency and reducing administrative overhead.
* Facilitate seamless single sign-on experiences by managing user identities and authentication across Elfsquad and other integrated applications, enhancing security and user convenience.

[Manage Users](/docs/apis/scim-api)
